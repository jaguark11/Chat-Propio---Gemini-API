# Chat-Propio---Gemini-API

Sistema de chat multimodal integrado con la API de Google Gemini (2.5 Flash).

## Características
* **Resiliencia:** Implementación de modo offline para fallos de API externa.
* **Multimodal:** Soporte para texto y flujos de audio serializados en Base64.
* **Persistencia:** Almacenamiento determinista en formato JSON.

## Instalación y Uso
1. **Clonar:** git clone https://github.com/jaguark11/Chat-Propio---Gemini-API.git
2. **Entorno:** python -m venv venv
3. **Activar:** .\venv\Scripts\activate
4. **Dependencias:** pip install flask requests
5. **Ejecutar:** python app.py
